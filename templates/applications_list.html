Security Features in This View
Role-Based Access Control (RBAC)

Branch officers see only their branch's applications. HO officers see all branches. Prevents horizontal privilege escalation.

SQL Injection Prevention

Search uses SQLAlchemy ORM with parameterized queries. User input is never concatenated into SQL.

Output Encoding

All user-generated content (names, remarks) is auto-escaped by Jinja2, preventing XSS attacks.

Audit Trail

Every application tracks who created it and when (created_by, created_at fields). Essential for compliance and forensics.
